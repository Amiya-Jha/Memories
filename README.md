# Memories 
User Authentication, Make a post, Delete your post, Add profile picture, Change username or name or bio, Like a post, Logout, Search other exising users.

1)Install node if you haven't.
2)Download essential packages within Memories folder by "npm i passport passport-local passport-local-mongoose mongoose express-session multer uuid"
3)make a file eg: users.js in route folder and set up your database.
4) "npx nodemon" on your terminal to run the webapp. (Now visit "localhost:Your port number for eg: "localhost:3000" on your browser)
5)View the web app in Mobile view (Inspect + turn on "Toggle Device Toolbar")
